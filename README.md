Sales Data Analysis Project

Description

This project analyzes sales data to generate insights such as total revenue and top-performing products.

Dataset Columns

- Order ID
- Product
- Category
- Price
- Quantity
- Date

Analysis Performed

- Calculated total revenue (Price × Quantity)
- Identified top-selling products
- Analyzed revenue by category
- Grouped and aggregated data using Pandas

Tools Used

- Python
- Pandas

Output

Generated insights from sales dataset# sales-data-analysis
